# Objetivos

<Info>
  Crear un entorno estable, accesible y económico para compartir, jugar y almacenar datos entre

  varios dispositivos.
</Info>

<Tabs>
  <Tab title="Aprendizaje">
    Aprender sobre Docker, redes privadas virtuales y administración de servidores.
  </Tab>
  <Tab title="Accesibilidad">
    Permitir que cualquier usuario conectado a Tailscale pueda acceder, subir o visualizar archivos.
  </Tab>
  <Tab title="Entretenimiento">
